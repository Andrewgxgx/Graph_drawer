# Quick Start Guide
Requirements 
1. Have Python installed on ur coompooterr

Start!
---
1. Firstly, create a virtual env to run the script, so it doesnt fuck up ur system wide python config

 **on mac/unix/linux**

```python
python3 -m venv venv
source ./venv/bin/activate
```


**on Windows**

```python
python -m venv venv
./venv/Scripts/activate
```

2. Then simply install the requirements

**On Windows / Mac / Unix / Linux**

```python
pip install -r requirements.txt
```

3. Excute the script 

**On Mac / Unix / Linux**

```python
python3 draw.py
```

**On Windows**

```python
python draw.py
```
